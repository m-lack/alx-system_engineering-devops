#!/bin/bash

101-rot13:
- This script applies the ROT13 encryption algorithm to text input, which is read from standard input.

102-odd:
- This script prints every other line from the input, starting with the first line.

103-water_and_stir:
- This script adds two numbers in custom bases "water" and "stir" stored in the `WATER` and `STIR` environment variables.
- It converts the custom base numbers to base 10 using string manipulation, adds them, and converts the result to another custom base "bestchol" before printing it.

100-decimal_to_hexadecimal:
- This script converts a decimal number stored in the `DECIMAL` environment variable to hexadecimal and displays the result.

13-print_float:
- This script prints a number with two decimal places, followed by a newline, stored in the `NUM` environment variable.

12-combinations:
- This script generates and prints all possible combinations of two lowercase letters from 'a' to 'z' in alphabetical order, excluding 'oo'.

11-binary_to_decimal:
- This script converts a binary number stored in the `BINARY` environment variable to decimal and displays the result.

10-love_exponent_breath:
- This script calculates and prints the result of raising a number stored in the `BREATH` environment variable to the power stored in the `LOVE` environment variable.

9-divide_and_rule:
- This script calculates and prints the result of dividing a number stored in the `POWER` environment variable by a number stored in the `DIVIDE` environment variable.
